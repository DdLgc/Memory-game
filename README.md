## Description

Memory Game est un jeu de mémoire développé en **HTML**, **CSS** et **JavaScript Vanilla**.

L'objectif est de retrouver toutes les paires de cartes en réalisant le moins de coups possible et dans le temps le plus court.

Ce projet m'a permis de consolider mes compétences en **JavaScript**, notamment sur la manipulation du DOM, la gestion des événements, les animations CSS, ainsi que la mise en place d'un **workflow Git professionnel** utilisant des branches de fonctionnalités (*feature branches*) et les **Conventional Commits**.

---

## Fonctionnalités

### Gameplay

- Génération dynamique des cartes
- Duplication automatique des cartes
- Mélange aléatoire des cartes
- Retournement des cartes
- Détection des paires
- Blocage des clics pendant la comparaison
- Animation de distribution des cartes
- Chronomètre
- Compteur de coups
- Fenêtre de victoire
- Affichage des statistiques de fin de partie
- Rejouer sans recharger la page

### Interface

- Interface responsive
- Animations CSS
- Effet de retournement des cartes
- Animation d'apparition de la fenêtre de victoire

## Installation

#### 1. Cloner le projet

```bash

#### 2. Ouvrir le projet

Ouvrir le dossier dans votre éditeur de code favori.

Ouvrir simplement le fichier :

```

## Workflow Git

Le développement du projet a été réalisé en utilisant un workflow Git proche des pratiques professionnelles.

### Types de branches

- `feature/*`
- `fix/*`
- `refactor/*`
- `docs/*`

### Conventional Commits

Exemples de commits utilisés :

```text
feat: initialize game board
feat: add shuffle function
feat: implement game timer
feat: implement win screen with game stats
feat: add move counter
feat: animate victory modal
feat: improve mobile responsiveness
refactor: clean up game logic and styles
docs: add project README
```
